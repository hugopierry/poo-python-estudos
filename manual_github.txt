Como subir código no github: # MANUAL RÁPIDO - PUBLICAR CÓDIGOS NO GITHUB

## PRIMEIRA VEZ (já foi feita)

Criar repositório local:

git init

Adicionar arquivos:

git add .

Criar commit:

git commit -m "Primeiro commit"

Conectar ao GitHub:

git remote add origin URL_DO_REPOSITORIO

Renomear branch:

git branch -M main

Enviar para o GitHub:

git push -u origin main

---

## PRÓXIMOS CÓDIGOS

Após alterar ou criar novos arquivos:

1. Verificar arquivos alterados:

git status

2. Adicionar alterações:

git add .

3. Criar novo commit:

git commit -m "Descrição da alteração"

Exemplos:

git commit -m "Adicionada herança em POO"

git commit -m "Exercício de listas"

git commit -m "Correção de bugs"

git commit -m "Novo projeto Python"

4. Enviar para o GitHub:

git push

---

## COMANDOS MAIS USADOS

Ver status:

git status

Ver histórico de commits:

git log

Enviar alterações:

git push

Baixar atualizações:

git pull

---

## FLUXO PADRÃO

Sempre que terminar um projeto:

git add .

git commit -m "Descrição do que foi feito"

git push

Pronto! O GitHub será atualizado.
